LiDAR + Sensors

Waymo, Luminar, Hesai, Velodyne — precise 3D mapping with laser sensors for centimeter-level accuracy.

0 believers

  • Waymo has completed 10M+ fully autonomous rides with zero driver fatalities
  • LiDAR provides precise 3D depth data — no AI guesswork needed for distance
  • Works in darkness, rain, fog — conditions where cameras struggle
  • LiDAR costs dropped from $75,000 to under $500 per unit in 10 years
  • Regulators trust sensor redundancy — easier path to approval
My conclusion: LiDAR wins
👁

Pure Vision

Tesla FSD, Comma.ai — cameras + neural networks only. If humans drive with eyes alone, why can't AI?

0 believers

  • Tesla has 6M+ vehicles collecting real-world driving data — no competitor is close
  • Camera-only costs $200/vehicle vs $2,000+ for LiDAR stack — 10x cheaper at scale
  • End-to-end neural networks improving exponentially with more data
  • Humans drive with vision alone — proving the problem is solvable without LiDAR
  • Elon Musk: "LiDAR is a crutch" — scaling pure vision is the harder but better path
My conclusion: Pure Vision wins
🛰

Sensor Fusion

Mobileye, NVIDIA DRIVE, Aurora — cameras + LiDAR + radar working together, each covering the others' blind spots.

0 believers

  • Mobileye's SuperVision deployed in millions of cars — proven at scale
  • Redundancy is a safety requirement — no single sensor type covers all conditions
  • NVIDIA DRIVE Orin/Thor platform supports all sensor types simultaneously
  • Aurora's commercially deployed autonomous trucks use multi-sensor fusion
  • Insurance and regulators prefer maximum sensor coverage for liability reasons
My conclusion: Sensor Fusion wins
